TTYCP(1)

НАЗВАНИЕ
ttycp - копирование между последовательными портами

СИНТАКСИС 

	ttycp  [источник [приемник]]

ОПИСАНИЕ
Команда ttycp копирует информацию из указанного файла-источника (по умолчанию - со стандартного ввода) в файл-приемник (по умолчанию на стандартный вывод). И источник, и приемник могут быть как последовательным интерфейсом, так и обычным файлом. В противоположность cp(1) команда ttycp позволяет организовать "наглядное" копирование данных между последовательными портами.

ПРИМЕРЫ

  1. Копировать ввод из /dev/tty1 в файл test.out: 
         ttycp  /dev/tty1 test.out
  2. Передать данные из файла test.out в /dev/tty2: 
         ttycp  test.out /dev/tty2
  3. Передать данные из /dev/tty1 в /dev/tty2: 
         ttycp  < /dev/tty1 > /dev/tty5

ФАЙЛЫ 

	/dev/tty*       Последовательные порты.

СМ. ТАКЖЕ
tm(1).

СЮРПРИЗЫ
Необходимо обладать правами, достаточными для открытия файла /dev/ttyномер_порта.
