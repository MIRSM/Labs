IPCRM(8)

НАЗВАНИЕ
ipcrm - удаляет идентификаторы очередей сообщений, семафоров или общей памяти.

СИНТАКСИС 

 ipcrm [options]

ОПИСАНИЕ
ipcrm - удаляет идентификаторы очередей сообщений, семафоров или общей памяти. Эти идентификаторы задаются следующими опциями:

-q msqid
удаляет идентификатор очереди сообщений msqid из системы и разрушает связанные с ним очередь сообщений и структуру данных.
-m shmid
удаляет из системы идентификатор общей памяти shmid. Связанные с ним сегмент общей памяти и структура данных разрушаются после последнего обращения.
-s semid
удаляет из системы идентификатор семафора semid и разрушает связанные с ним набор семафоров и структуру данных.
-Q msgkey
удаляет идентификатор очереди сообщений, созданный с помощью ключа msgkey, из системы и разрушает связанные с ним очередь сообщений и структуру данных.
-M shmkey
удаляет из системы идентификатор общей памяти, созданный с помощью ключа shmkey. Связанные с ним сегмент общей памяти и структура данных разрушаются после последнего обращения.
-S semkey
удаляет из системы идентификатор семафора, созданный с помощью ключа semkey и разрушает связанные с ним набор семафоров и структуру данных.

Подробности этих удалений описываются на страницах msgctl(2), shmctl(2) и semctl(2). Можно найти упомянутые идентификаторы и ключи с помощью ipcs(8).

СМ. ТАКЖЕ
ipcs(8), msgctl(2), msgget(2), msgop(2), semctl(2), semget(2), semop(2), shmctl(2), shmget(2), shmop(2)

ПРИМЕЧАНИЕ
ipcrm не может использоваться для удаления семафоров, созданных с помощью creatsem(2), или для удаления общей памяти, созданной с помощью sdget(2).
