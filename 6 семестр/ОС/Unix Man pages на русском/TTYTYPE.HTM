TTYTYPE(4)

НАЗВАНИЕ
ttytype - информация о типах терминалов

ОПИСАНИЕ
В файле /etc/ttytype содержатся определения типов терминалов для специальных файлов из каталога /dev. При входе в систему выполняется файл /etc/profile. Этот файл содержит последовательность команд: 

     ta=`tty`
     ta=`grep "$ta " /etc/ttytype`
     TERM=`expr "$ta" : '.* \(.*\)' \| "$ta"`

в результате выполнения которой устанавливается переменная окружения TERM. Программы, использующие специальные возможности терминала [например, vi(1)], читают значение переменной TERM и используют его в качестве ключа для базы данных termcap или terminfo(4).

ПРИМЕР
Строка в файле /etc/ttytype: 

     /dev/console vt100
определяет консоль как терминал типа vt100.

СМ. ТАКЖЕ
term(4), terminfo(4), term(5).
