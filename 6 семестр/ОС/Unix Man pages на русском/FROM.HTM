FROM(1)
FreeBSD Reference Manual

ИМЯ
from - печатает имена тех, от кого пришла почта

СИНТАКСИС 

 from [-s sender] [-f file] [user]

ОПИСАНИЕ
From печатает все строки с заголовком from: из почтового ящика

ОПЦИИ
-f file Использовать файл file вместо почтового ящика пользователя, вызвавшего from. Если опция -f используется, аргумент user не может присутсвовать. Если имя файла присутсвует - читает со стандартного ввода.
-s sender Печатается только почта от указанных адресатов в строке sender.

Если аргумент user присутсвует, то читается его почтовый ящик, а не почтовый ящик пользователя, вызвавшего from. (Только для суперпользователя).

ФАЙЛЫ 

 /var/mail/*
