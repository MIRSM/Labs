HYPOT(3M)

НАЗВАНИЕ
hypot, hypotf, cabs, cabsf - функция, вычисляющая эвклидово расстояние

СИНТАКСИС 

	#include <math.h>
	
	double hypot (x, y)
	double x, y;
	
	float
	hypotf(float x, float y)
	
	struct {double x, y;} z;
	
	double
	cabs(z)
	
	struct {float x, y;} z;
	
	float
	cabsf(z)

ОПИСАНИЕ
Функция hypot возвращает значение 

     sqrt (x*x + y*y)
которое вычисляется по методу, исключающему "паразитные" переполнения.

СМ. ТАКЖЕ
matherr(3M).

ДИАГНОСТИКА
Если к переполнению приводит попытка представления корректного результирующего значения, то функция hypot возвращает значение HUGE [см. intro(3)], а внешней переменной errno присваивается значение ERANGE.

Изложенная процедура обработки ошибок может быть изменена посредством функции matherr(3M)
