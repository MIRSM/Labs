DISABLE(1)

ИМЯ
disable - Отключает терминалы и принтеры.

СИНТАКСИС 

 disable tty ...
 disable [-c][-r[reason]]printers

ОПИСАНИЕ
Что касается терминалов, эта программа манипулирует файлом /etc/ttys и выдает сигнал команде init на запрещение вхождения в систему с некоторого терминала. Что касается принтеров, команда disable останавливает требования на печать, отправленные к заданному параметром printers принтеру. Могут быть использованы следующие опции:
-c Отменяет всевозможные требования, которые в данный момент печатаются.
-r[reason] Связывает значение параметра reason (причина) с отключением данного принтера. Параметр reason применяется ко всем принтерам из списка printers, который следует за опцией -r. Если опция -r не указана, или опция -r задана без параметра reason, то используется значение reason, назначенное по умолчанию. Значение reason сообщается командой lpstat(1).

ПРИМЕРЫ
В этом примере принтер с именем linepr блокируется из-за сбоя бумаги ("paper jam"): 

 disable -r"paper jam" linepr

ФАЙЛЫ 

 /dev/tty*
 /etc/ttys
 /usr/spool/lp/*
