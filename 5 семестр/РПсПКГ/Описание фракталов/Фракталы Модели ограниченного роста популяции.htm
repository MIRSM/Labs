Фракталы: Модели ограниченного роста популяции

program Popul;
Uses CRT, Graph;
Var
   gd,gm: Integer;
   x, a: Real;
Begin
   gd := Detect;
   InitGraph(gd,gm,'c:\bp\bgi');
   x:=0.6;
   a:=2.7;
   Repeat
     x:=a*x*(1-x);
     PutPixel(Round(x*640),Round((a-2.8)*400),1);
     a:=a+0.000001;
   Until a>=4;
   ReadKey;
   CloseGraph;
End.

Можно использовать следующие модели:

f = a*x(1-x)
f = a*sin(pi*x)
f = x*exp(a*(1-x))
f = x*(1+a*(1-x))
f = a*x/(1+a*x)5
