Фракталы: Martin, Hopalong, Chip, Quadruptwo, Threeply

Martin

При построении используются следующие формулы:
x'=y-sin(x)
y'=a-x,
где a, например, равен 3.14. Следующая программа строит этот фрактал. 
Program Martin;
Uses Graph, Crt;
Var
   x, t, y  : Real;
   n        : Integer;
   gd, gm   : Integer;
Begin
   gd := Detect;
   InitGraph(gd,gm,'e:\bp\bgi');
   x:=0;
   y:=0;
   n:=0;
   While Not KeyPressed Do Begin
     t:=x;
     x:=y-sin(x);
     y:=3.14-t;
     inc(n);
     PutPixel(round(x*2)+320,round(y*2)+240,n mod 16)
   End;
   Readkey;
   CloseGraph;
end.

Hopalong

Здесь используются формулы вида:
x'=y-sign(x)*sqrt(abs(b*x-c))
y'=a - x
Примеры параметров:
a=0.4, b=1,с=0.
a=1, b=4,с=60.
a=-1, b=-2,с=-3.
a=-1000, b=0.1,с=-10.
Функция сигнала определена как sign=x/|x|

Chip

x'=y-sign(x)*cos(sqr(ln(abs(b*x-c))))
y'=a - x
a=-15, b=-19, c=1.

Quadruptwo

x'=y-sign(x)*sin(ln(abs(b*x-c)))*arctan(sqr(ln(abs(c*x-b))))
y'=a-x
a=34, b=1, c=5

Threeply

x' = y-sign(x)*(abs(sin(x)*cos(b)+c-x*sin(a+b+c)))
y' = a - x
a=-55, b=-1, c=-42
