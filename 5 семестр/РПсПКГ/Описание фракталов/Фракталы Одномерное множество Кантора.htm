Фракталы: Одномерное множество Кантора

Один из основателей теории множеств, Кантор, придумал один из старейших фракталов (1883). Этот фрактал строится следующим образом. Из исходного отрезка единичной длины выбрасывается интервал (1/3, 2/3). Далее операция повторяется, и в пределе мы получим фрактал Кантора. На западе подобные множества называют пылью Кантора.

Следующая программа на языке Pascal создает изображение одномерного множества Кантора рекурсивным методом.

Program Cantor1;
Uses CRT, Graph;
Const
  min = 1;
Var
  gd, gm : Integer;

Procedure Draw(x, y : Real; Size : Real);
  Var
    s : Real;
Begin
  If size > min then Begin
    s := size / 3;
    Draw(x, y + 20, s);
    Draw(x + s * 2, y + 20, s);
  End;
  Bar(Round(x), Round(y), Round(x + size), Round(y + 5));
End;

Begin
  gd := Detect;
  InitGraph(gd,gm,'');
  Draw(10,30,500);
  ReadKey;
  CloseGraph;
End.

Не забудте в процедуре инициализации графического режима указать путь к графическому драйверу!

Существует еще одна разновидность одномерного множества Кантора, называемая гребень Кантора. Для ее построения нужно в предыдущей программе строчку 

Bar(Round(x), Round(y), Round(x + size), Round(y + 5));
заменить на 
Rectangle(Round(x), Round(y), Round(x + size), Round(y + 20));.
