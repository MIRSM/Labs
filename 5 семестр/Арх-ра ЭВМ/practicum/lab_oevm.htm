Целью лабораторного практикума является изучение принципов работы подсистем современных ЭВМ: процессора, памяти периферийных устройств.

Технической базой лабораторного практикума является IBM совместимая персональная ЭВМ с операционной системой Windows 98/XP/2000. В качестве инструментального средства для изучения функционирования ЭВМ используется язык Ассемблер. Все необходимое программное и информационно справочное обеспечение содержится в настоящем электронном пособии.

Отчеты по выполненным лабораторным должны содержать:

 

Название дисциплины,
Название лабораторной работы,
Номер группы,
Цель работы,
Задание (с указанием номера варианта),
Листинг программы с комментариями (после компиляции),
Результаты решения .

К отчету представляются исходный текст программы и исполняемый модуль на электронном носителе.

Лабораторная работа 1 "Изучение работы с Турбо-ассемблером и Турбо-отладчиком"

Цель работы: изучение технологии разработки, компиляции и отладки программ на языке Ассемблер.

Задание: разработать и отладить программу вычисления выражения Y=A+B-C, где все операнды - 16 разрядные числа (слова), размещенные в памяти. Для выполнения задания использовать готовую программу в файле prim.asm. Этот файл рекомендуется использовать в качестве шаблона при выполнении последующих лабораторных работ.

Порядок выполнения работы:

1. Создайте на одном из дисков Вашего компьютера папку …/oevm, в которой будут размещаться используемое программное обеспечение и разрабатываемые программы.
2. Скопируйте в эту папку файлы prim.zip и tasm.zip и распакуйте их.
3. Изучите теоретический материал по структуре программы на Ассемблере ( разделы 2, 3.1-3.3 и 4.1 методических указаний ПРОГРАММИРОВАНИЕ НА АССЕМБЛЕРЕ ПЭВМ).Разберите программу, представленную в файле prim.asm, используя Справочную систему по языку ассемблера IBM PC.
4. Следуя указания, приведенным в методичке, выполните компиляцию, компоновку и отладку программы.
5. Используя Справочную систему по языку ассемблера IBM PC, изучите управление процессами компиляции и компоновки с помощью ключей.

Лабораторная работа 6. "Таймер и звук"

Цель работы: изучить принцип управления системным таймером и способы генерации звука.

Задание: Разработать и отладить программу генерации звукового сигнала. Генерация должна прекращаться с нажатием любой клавиши.

Порядок выполнения работы:

1. Изучить материал по работе с таймером и звуком.  ( дополнительный материал)
2. Написать и отладить программу.
3. Модернизировать программу, обеспечивая изменение тона сигнала при каждом нажатии клавиши. Определить с помощью модернизированной программы значения коэффициента пересчета таймера, при которых обеспечивается высший и низший тон услышанного звукового сигнала.

Цель работы: изучить организацию гибкого магнитного диска на физическом и логическом уровнях, а так же ввода и вывода данных на гибкий диск.

Задание 1. Исследовать логическую организацию гибкого магнитного диска

Порядок выполнения работы:

3. Загрузить и распаковать программу Disk Editor (файл de.zip)
4. Используя программу Disk Editor: ·
найти и разобрать формат BOOT - записи,
· найти область корневого каталога и разобрать формат записи корневого каталога, соответствующей одному из записанных файлов,
